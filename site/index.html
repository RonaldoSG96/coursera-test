Related to Browser Sync

To open browser sync, the required steps are:
  1. in your command prompt, use the dir and cd commands to navigate to your local repository
  2. type browser-sync start --server --directory --files "*"

Related to Git

More useful commands:
  • 'git status' (checking the status of the code)
  • 'git add .' (if I recall correctly the dot means you add everything there is to be added)
  • 'git commit -m"type your message here"' (with this you commit whatever you have previously added)
  • 'git push' (with this you push your committed local files to the online server, thus making it available online)
